# 엑셀

- 문자 왼쪽정렬
    - '__' : 문자
    - ###  : 자리 모자름
    - alt+enter 

- 숫자 오른쪽정렬
    - 공백,특수문자 -> 문자인식
    - 음수 : - , ()
    - 분수 : 0_1/2

- 날짜 오른쪽 정렬
    - 연도 두자리 : 30이상 1900  29이하 2000년대
    - 오늘날짜 : Ctrl + ;
    - 현재시간 : Ctrl + shift +;

- 메모 shift + f2
    - 데이터를 지워도 메모는 남아있다.

- 윗주 : 문자데이터 삽입
    - 윗주 삽입,수정 -> 윗주편집
    - 윗주 글꼴,맞춤 -> 윗주설정
    - 서식은 전체적용

- 채우기 핸들
- 데이터수정 
    - ctrl+enter : 전체수정
    - double click or f2 : 부분수정
